# ZOMATO-INSIGHTS-DASHBOARD-PWI
This project presents an interactive Power BI dashboard built using Zomato dataset to analyze restaurant trends, customer preferences, and market insights. It helps users explore key metrics such as ratings, cuisines, pricing, and location-based performance to support data-driven decision-making.
This project is a data analytics dashboard built using Microsoft Excel and Power BI to analyze Zomato restaurant data. It provides meaningful insights into restaurant trends, customer preferences, ratings, and pricing patterns.

The goal of this project is to transform raw data into interactive visualizations that help in better decision-making and understanding market trends.

🚀 Features
📍 City-wise restaurant distribution
⭐ Ratings analysis
🍴 Cuisine popularity insights
💰 Cost distribution & pricing trends
📈 Interactive dashboards with filters & slicers
🔍 Data cleaning and transformation using Excel
🛠️ Tools & Technologies Used
Microsoft Excel
Data Cleaning
Data Preprocessing
Power BI
Data Visualization
Dashboard Creation


📌 Key Insights
Most restaurants are concentrated in major cities
Certain cuisines dominate specific regions
Higher ratings are often associated with moderate pricing
Customer preferences vary significantly by location
